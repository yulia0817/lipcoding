# Team Mode Implementation Plan (플랜-팀모드)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a team mode ("함께 타는 모닥불") where people join a team by code, their focus sessions fuel one shared campfire, and they compete/cooperate via a live leaderboard, team goal, presence, ember feed, and team-level gamification (team level, MVP badge, streak).

**Architecture:** A new in-memory team store creates/joins teams by short code and tracks members by the same anonymous `X-User-Id` used elsewhere plus a display name. Sessions gain optional `user_id`/`user_name`/`team_id` fields (backward compatible) so existing per-user and new per-team aggregations both work. A lightweight presence map (heartbeat with TTL) shows who is focusing now. The frontend polls team summary/presence every few seconds (no WebSockets — YAGNI) and reuses the themeable `FocusScene` for the shared team campfire.

**Tech Stack:** FastAPI + Pydantic v2 / Python 3.11+ (backend), React 18 + Vite (frontend), pytest + httpx (backend tests).

## Global Constraints

- Anonymous identity: header `X-User-Id` (localStorage `fc_uid`); display name stored client-side in `localStorage['fc_name']` and sent in request bodies.
- Storage in-memory (`team_store.py`), mirroring the `Lock` + dict pattern of `session_store.py`. No database.
- All additions are backward compatible: existing `/api/sessions`, `/api/stats`, etc. keep working with no `team_id`.
- Team codes: 6-char uppercase alphanumeric, unambiguous (no `0/O/1/I`).
- Presence TTL: a member counts as "focusing now" if their last heartbeat was within 60 seconds.
- Polling interval (frontend): 5 seconds for team summary + presence. Stop polling when the tab is hidden.
- Reuse `frontend/src/design` components and `FocusScene` (from 플랜-스킨상점) for the team campfire. Korean UI copy.
- Backend tests run from `backend/`: `backend/venv/bin/python -m pytest`.
- One commit per task. Commit format `[영역] 메시지`.

- [ ] **Step 4: Create the store**

Create `backend/team_store.py`:

```python
import secrets
import uuid
from threading import Lock
from typing import Dict, Optional

from team_models import Member, Team

CODE_ALPHABET = "ABCDEFGHJKLMNPQRSTUVWXYZ23456789"  # no 0/O/1/I


class TeamStore:
    def __init__(self) -> None:
        self._teams: Dict[str, Team] = {}
        self._by_code: Dict[str, str] = {}
        self._lock = Lock()

    def _gen_code(self) -> str:
        while True:
            code = "".join(secrets.choice(CODE_ALPHABET) for _ in range(6))
            if code not in self._by_code:
                return code

    def create(self, name: str, owner_id: str, owner_name: str) -> Team:
        with self._lock:
            code = self._gen_code()
            team = Team(
                id=uuid.uuid4().hex,
                code=code,
                name=name,
                members=[Member(user_id=owner_id, name=owner_name)],
            )
            self._teams[team.id] = team
            self._by_code[code] = team.id
            return team

    def find_by_code(self, code: str) -> Optional[Team]:
        tid = self._by_code.get(code.upper())
        return self._teams.get(tid) if tid else None

    def join(self, code: str, user_id: str, user_name: str) -> Team:
        with self._lock:
            tid = self._by_code.get(code.upper())
            if not tid:
                raise ValueError("not_found")
            team = self._teams[tid]
            for m in team.members:
                if m.user_id == user_id:
                    m.name = user_name
                    return team
            team.members.append(Member(user_id=user_id, name=user_name))
            return team

    def get(self, team_id: str) -> Team:
        team = self._teams.get(team_id)
        if not team:
            raise ValueError("not_found")
        return team


team_store = TeamStore()
```

```python
    def summary(self, team_id, sessions, goal_minutes=120):
        team = self.get(team_id)
        member_names = {m.user_id: m.name for m in team.members}
        today = datetime.now(KST).date()

        today_min = 0
        total_min = 0
        per_user_min = defaultdict(int)
        per_user_cnt = defaultdict(int)

        for s in sessions:
            if s.team_id != team_id:
                continue
            total_min += s.duration_min
            if s.created_at.astimezone(KST).date() == today:
                today_min += s.duration_min
                uid = s.user_id or "anon"
                per_user_min[uid] += s.duration_min
                per_user_cnt[uid] += 1

        leaderboard = [
            LeaderRow(
                user_id=uid,
                name=member_names.get(uid, s_name or uid),
                minutes=per_user_min[uid],
                sessions=per_user_cnt[uid],
            )
            for uid, s_name in ((u, member_names.get(u)) for u in per_user_min)
        ]
        leaderboard.sort(key=lambda r: r.minutes, reverse=True)
        mvp_id = leaderboard[0].user_id if leaderboard else None

        fire_level = 0 if today_min <= 0 else min(5, 1 + (today_min - 1) // 60)
        goal_progress = min(1.0, today_min / goal_minutes) if goal_minutes else 0.0

        return TeamSummary(
            team_id=team.id,
            name=team.name,
            code=team.code,
            member_count=len(team.members),
            today_minutes=today_min,
            total_minutes=total_min,
            goal_minutes=goal_minutes,
            goal_progress=round(goal_progress, 3),
            fire_level=fire_level,
            leaderboard=leaderboard,
            mvp_id=mvp_id,
        )
```

```python
from datetime import datetime, timezone
from threading import Lock
from typing import Dict, Tuple

UTC = timezone.utc


class PresenceStore:
    def __init__(self) -> None:
        # (team_id, user_id) -> {name, task, at}
        self._beats: Dict[Tuple[str, str], dict] = {}
        self._lock = Lock()

    def beat(self, team_id, user_id, name, task, now=None) -> None:
        now = now or datetime.now(UTC)
        with self._lock:
            self._beats[(team_id, user_id)] = {"name": name, "task": task, "at": now}

    def live(self, team_id, now=None, ttl=60) -> list[dict]:
        now = now or datetime.now(UTC)
        with self._lock:
            rows = []
            for (tid, uid), v in self._beats.items():
                if tid != team_id:
                    continue
                seconds = (now - v["at"]).total_seconds()
                if seconds <= ttl:
                    rows.append({
                        "user_id": uid,
                        "name": v["name"],
                        "task": v["task"],
                        "seconds_ago": int(seconds),
                    })
        rows.sort(key=lambda r: r["seconds_ago"])
        return rows


presence_store = PresenceStore()
```

Create `backend/routers/teams.py`:

```python
from datetime import timedelta, timezone

from fastapi import APIRouter, Header, HTTPException
from pydantic import BaseModel

from presence_store import presence_store
from session_store import session_store
from team_models import Team, TeamSummary
from team_store import team_store

router = APIRouter(prefix="/api/teams", tags=["teams"])
KST = timezone(timedelta(hours=9))


def _uid(x_user_id):
    return x_user_id or "anon"


class CreateBody(BaseModel):
    name: str
    user_name: str = "익명"


class JoinBody(BaseModel):
    code: str
    user_name: str = "익명"


class BeatBody(BaseModel):
    user_name: str = "익명"
    task: str = ""


@router.post("", response_model=Team)
async def create_team(body: CreateBody, x_user_id: str | None = Header(default=None)) -> Team:
    return team_store.create(body.name, _uid(x_user_id), body.user_name)


@router.post("/join", response_model=Team)
async def join_team(body: JoinBody, x_user_id: str | None = Header(default=None)) -> Team:
    try:
        return team_store.join(body.code, _uid(x_user_id), body.user_name)
    except ValueError:
        raise HTTPException(status_code=404, detail="not_found")


@router.get("/{team_id}/summary", response_model=TeamSummary)
async def team_summary(team_id: str) -> TeamSummary:
    try:
        return team_store.summary(team_id, session_store.list())
    except ValueError:
        raise HTTPException(status_code=404, detail="not_found")


@router.post("/{team_id}/beat")
async def beat(team_id: str, body: BeatBody, x_user_id: str | None = Header(default=None)) -> dict:
    presence_store.beat(team_id, _uid(x_user_id), body.user_name, body.task)
    return {"ok": True}


@router.get("/{team_id}/presence")
async def presence(team_id: str) -> list[dict]:
    return presence_store.live(team_id)


@router.get("/{team_id}/feed")
async def feed(team_id: str) -> list[dict]:
    rows = []
    for s in session_store.list():  # already newest-first
        if s.team_id == team_id and s.retro:
            rows.append({
                "date": s.created_at.astimezone(KST).date().isoformat(),
                "name": s.user_name or "익명",
                "task": s.task,
                "retro": s.retro,
            })
        if len(rows) >= 30:
            break
    return rows
```

```javascript
import { useState } from 'react'
import { api } from '../../api'

export function TeamGate({ onEntered }) {

  )
}
```

```javascript
import { useEffect, useRef, useState } from 'react'
import { api } from '../../api'
import { Button, Card, EmptyState, useToast } from '../../design'
import { FocusScene } from '../../focus/FocusScene'
import { getUserId, setTeamId } from '../../lib/identity'
import './team.css'

const RANK_EMOJI = ['🥇', '🥈', '🥉']

export function TeamDashboard({ teamId, onLeave }) {
  const [summary, setSummary] = useState(null)
  const { toast } = useToast()
  const timerRef = useRef(null)
  const me = getUserId()

  async function load() {
    try {
      setSummary(await api.teamSummary(teamId))
    } catch (e) {
      toast(`팀 정보를 불러오지 못했어요: ${e}`, { variant: 'error' })
    }
  }

  useEffect(() => {
    load()
    timerRef.current = setInterval(() => {
      if (!document.hidden) load()
    }, 5000)
    return () => clearInterval(timerRef.current)

    setTeamId(null)
    onLeave()

  const fire = '🔥'.repeat(summary.fire_level) || '🪵'

  return (
    <>
      <Card>
        <div className="team-fire">
          <FocusScene skinId="campfire" intensity={summary.goal_progress} active={summary.today_minutes > 0} />
          <div className="team-fire__big">{fire}</div>
          <div className="section-title" style={{ marginTop: 8 }}>{summary.name}</div>
          <div className="team-code">{summary.code}</div>
          <p className="app__sub">멤버 {summary.member_count}명 · 오늘 {summary.today_minutes}분</p>
        </div>
      </Card>

      <div className="section-title">오늘의 팀 목표 ({summary.today_minutes}/{summary.goal_minutes}분)</div>
      <Card>
        <div className="goal-track">
          <div className="goal-fill" style={{ width: `${summary.goal_progress * 100}%` }} />
        </div>
      </Card>

      <div className="section-title">리더보드 (오늘)</div>
      <Card>
        {summary.leaderboard.length === 0 ? (
          <p className="app__sub" style={{ margin: 0 }}>아직 오늘 집중한 멤버가 없어요.</p>
        ) : (
          summary.leaderboard.map((row, i) => (
            <div className="leader-row" key={row.user_id}>
              <span className="leader-row__rank">{RANK_EMOJI[i] || i + 1}</span>
              <span className="leader-row__name">
                {row.name}{row.user_id === me ? ' (나)' : ''}
                {row.user_id === summary.mvp_id ? ' 👑' : ''}
              </span>
              <span className="leader-row__min">{row.minutes}분 · {row.sessions}세션</span>
            </div>
          ))
        )}
      </Card>

      <div style={{ marginTop: 16 }}>
        <Button variant="secondary" onClick={leave}>팀 나가기</Button>
      </div>
    </>
  )
}
```
