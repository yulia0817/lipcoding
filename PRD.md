# PRD — Focus Scene (포커스 신)

> 천하제일 입코딩 대회 출품작 · 개인 생산성 앱 · 음성 기반 포모도로 집중 앱

## 1. 한 줄 소개

혼자서도, 친구와 **같이 집중**해도 매일의 집중이 시각적 보상과 기록으로 쌓입니다.

> ④ 집중할 일을 입력하고 타이머 시작.

## 2. 문제 정의

- 집중을 시작하는 마찰이 크다 — 타이머를 켜고 무엇에 집중할지 정하는 과정 자체가 귀찮다.
- 집중이 추상적이라 동기부여가 약하다 — 내가 얼마나 집중했는지 체감되지 않는다.
- 혼자 하는 집중은 쉽게 무너진다 — 함께하는 사람이 있으면 더 오래 집중한다.

## 3. 목표 사용자

- 포모도로 기법으로 공부·작업하는 학생/직장인
- 손을 떼지 않고(음성으로) 빠르게 집중을 시작하고 싶은 사용자
- 친구와 가볍게 집중 챌린지를 하고 싶은 사용자

## 4. 핵심 가치 / 차별점

1. **음성 입력으로 즉시 시작** — "보고서 작성"이라고 말하면 바로 집중 세션 시작 (Web Speech API, 한국어).
2. **살아있는 모닥불 메타포** — 집중하면 불꽃이 커지고, 멈추면 사그라든다. 추상적 집중을 시각화.
3. **혼자 → 같이** — 초대 코드로 친구를 모아 각자의 집중 시간을 합산해 공동 목표를 채운다.

## 5. 핵심 기능 (Source of Truth)

### 집중 타이머
- 음성 또는 텍스트로 집중 대상 입력 후 포모도로 시작.
- 프리셋(15/25/45/50분) 또는 직접 입력. **세션 중에도 시간 변경 시 남은 시간 자동 조정.**
- 진행에 따라 모닥불 불꽃이 커지는 CSS 애니메이션. 몰입(전체화면) 모드 제공.
- 세션 종료 시 음성/텍스트로 **한 줄 회고** 기록.

### 휴식 모드
- 집중 완료 시 자동으로 휴식 모드 전환.
- 앉아서 하는 가벼운 스트레칭을 안내하고 음성(ko-KR)으로 읽어줌.

### 기록 & 분석
- **불씨 저널**: 이번 주 집중 요약(총 집중 시간·불 세기·키워드)과 회고 모음.
- **날짜별 기록**: 날짜별로 무엇에 얼마나 집중했는지 타임라인/막대로 시각화.
- **통계**: 오늘 집중 시간, 연속 집중일(streak), 완료 세션 수, 12주 히트맵, 집중 vs 딴짓 비율.

### 보상 (게이미피케이션)
- 레벨/XP/장작(코인), 일일 퀘스트, 배지, 연속 집중(streak).
- **스킨 보관함**: 모닥불 외 벽난로/해변/숲/비/우주/오로라 등 테마를 자유롭게 선택.

### 로그인 & 테스터 모드
- 자체 회원가입/로그인(아이디·비밀번호, pbkdf2 해싱).
- **"테스터로 입장"**: 회원가입 없이 더미 데이터로 모든 기능을 즉시 체험.

### 같이 집중 (소셜)
- 초대 코드(6자리)로 친구를 모아 모임 생성/참여 (최대 8명).
- 각자의 개인 집중 시간이 합산되어 **공동 목표**를 채움(인당 60분 기준).
- 실시간 접속 현황(누가 무엇에 집중 중인지), 리더보드/MVP, 활동 피드.

## 6. 사용자 흐름 (E2E)

1. 첫 화면에서 **로그인** 하거나 **테스터로 입장**.
2. 집중 대상을 음성/텍스트로 입력하고 타이머 시작 → 모닥불이 타오름.
3. 완료 시 한 줄 회고 작성 → 휴식 모드로 스트레칭.
4. 통계/날짜별 기록/불씨 저널에서 집중 데이터를 확인하고 보상 획득.
5. (선택) **같이 집중**에서 초대 코드로 친구와 공동 목표에 도전.

## 7. 기술 스택

- **프론트엔드**: React 18 + Vite, 자체 디자인 시스템(라이트 테마), Web Speech API.
- **백엔드**: Python FastAPI (인메모리 저장), 사용자별 세션 분리, REST API.
- **배포**: Azure Container Apps (azd로 backend/frontend 컨테이너 배포), Korea Central.

|--------|------|------|

| GET  | `/api/stats` | 오늘 집중·streak·히트맵·집중비율 |

## 9. 배포 정보

- 프론트엔드(앱): Azure Container Apps (Korea Central).
- 백엔드 API: Azure Container Apps, `/api/*` 프록시.
- 배포 도구: `azd up` (리소스 그룹·Container Apps 환경 자동 프로비저닝).

## 10. 수용 기준 (Acceptance Criteria · 자동 평가 체크리스트)

각 항목은 배포 앱에서 실제로 검증 가능합니다.

- [ ] 첫 화면에 로그인 폼과 **"테스터로 둘러보기"** 버튼이 보인다.
- [ ] "테스터로 둘러보기"를 누르면 가입 없이 메인 앱(사이드바 + 집중 타이머)으로 진입한다.
- [ ] 집중할 일을 입력하고 시작하면 타이머가 동작하고 모닥불 애니메이션이 커진다.
- [ ] 사이드바에서 **집중 타이머 / 같이 집중 / 주간 리포트 / 날짜별 기록 / 활동 분석 / 통계 / 스킨 보관함**으로 이동된다.
- [ ] **통계** 화면에 오늘 집중 시간·연속 집중일(streak)·완료 세션 수·히트맵·집중 비율이 표시된다.
- [ ] **같이 집중**에서 "모임 만들기"로 초대 코드가 생성되고, "코드로 참여"로 다른 사용자가 합류할 수 있다.
- [ ] 같이 집중 모임에 공동 목표 게이지·리더보드·실시간 접속 현황이 표시된다.
- [ ] **스킨 보관함**에서 테마를 바꾸면 집중 화면 분위기가 바뀐다.
- [ ] 집중 완료 시 휴식 모드로 전환되어 스트레칭을 안내한다.
- [ ] 백엔드 `GET /api/stats`, `POST /api/auth/register`, `POST /api/teams` 등이 정상 응답한다.

## 11. 범위 밖 (Out of Scope)

- 영구 데이터베이스(현재 인메모리 — 재시작 시 초기화).
- 외부 OAuth(구글/깃헙) 로그인 — 자체 간단 로그인으로 대체.
- 모바일 네이티브 앱 — 반응형 웹으로 제공.
